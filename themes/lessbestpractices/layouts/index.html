LESS is a dynamic style sheet language that extends the capability of CSS.

LESS is what is known as a CSS preprocessor, something which compiles back into regular CSS, which can then be read by a browser.

LESS provides features like variables, functions, mixins and operations that allow you to write more structured styles with less code

Why do we use LESS?

  • LESS supports creating cleaner, cross-browser friendly CSS faster and easier.
  • LESS allows you to write modular CSS code, making it readable and easily maintainable.
  • It will make your CSS DRY (Don't repeat yourself).Don't repeat yourself is a core principal of computer programming.

Advantages

  • LESS easily generates CSS that works across all browsers
  • LESS enables you to write modular CSS code
  • LESS provides the use of mixins, variables and functions that enable us to save time and write better CSS code

Installation

The easiest way to install LESS is via npm, the node.js package manager as below:


npm install less

Please go through lesscss.org for more details on installation.
